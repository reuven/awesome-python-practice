# Contributing

Thanks for helping make **Awesome Python Practice** better! This list aims to
point people to the best resources for *practicing* Python.

## What belongs here

A resource is a good fit if it helps people **practice** Python — exercises,
challenges, katas, interactive platforms, problem sets, practice-oriented books,
or recurring challenge newsletters. Tutorials and reference docs that don't
involve *doing* are out of scope (there are other lists for those).

Both free and paid resources are welcome, as long as they're genuinely useful.

## How to add a resource

1. Fork the repo and edit `README.md`.
2. Add your entry to the **most appropriate existing section**, in this format:

   ```
   - [Name](https://example.com/) — A short, neutral, one-line description.
   ```

3. Keep descriptions factual and concise — no marketing language, no ALL CAPS,
   no "best ever." Say what it is and who it's for.
4. Check that the link works and uses `https` where available.
5. Keep entries roughly grouped by relevance within a section.
6. Open a pull request with a one-line summary of what you added and why.

## Guidelines

- **One resource per entry.** Don't bundle several links into one bullet.
- **No dead or parked links.** If a resource has gone offline, a PR removing it
  is just as welcome as one adding a new resource.
- **Disclose affiliation.** If you're the author/owner of a resource, that's
  fine — just mention it in the PR so reviewers know.
- **Quality over quantity.** A focused, trustworthy list is more useful than an
  exhaustive one.
